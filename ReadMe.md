



## Pandas Overview

Pandas is a powerful Python library for data manipulation and analysis. It provides data structures and functions needed to work with structured data seamlessly.
This repository shows the basics such as a dataframe,series ,importing csvs and reading them ,accessing elements using iloc() and loc() functions

## Features

- **DataFrames & Series**: Flexible data structures for organizing and analyzing data
- **Data Cleaning**: Tools for handling missing data and duplicates
- **Data Transformation**: Reshape, pivot, and merge datasets easily
- **Time Series**: Built-in support for time-indexed data
- **I/O Tools**: Read/write CSV, Excel, SQL, and more

## Installation

```bash
pip install pandas
```

## Quick Start

```python
import pandas as pd

# Create a DataFrame
df = pd.DataFrame({'Name': ['Alice', 'Bob'], 'Age': [25, 30]})

# Display data
print(df)
```
